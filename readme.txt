== Description ==

Plugin banning links to transmit PageRank domains recipients.
The plugin Seo Hide uses the redirect pages alternation javascript
It is recommended to use in order to avoid sanctions search engines related to the publication of external links to your site.
This plugin will hide all the links are in the text of the page or post.
